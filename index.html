About Project

Machine Learning on Rescue data

The limited capacity to deal with emergencies and disasters was putting the lives and properties of the citizens at risk. This was evident from the fact that there were over 95% chances that in case of emergency; the victim will not get an ambulance for transportation, let alone timely emergency care by trained professionals. We will be able to predict the number of accidents so that we prepare rescue team to take necessary steps could be taken to save the life of victim.

  • Data Gathering
  • Analysis
  • Preprocessing
  • Using Recurrent Neural Network
  • Using LSTM
  • Traing of Model
  • Predicting Number of Accidents
